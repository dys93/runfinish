# 1.项目说明

## 项目基本功能

本项目主要实现了 test03 的解析，能够正确解析的 markdown 语法最主要有：

* 标题 h1 到 hn 
* 3级列表ul ol
* strong标记
* em标记
* code标记
* pre code 代码块
* hr
* 连接a href
* gump link
* word link


## 测试文件和测试方法

```
测试文件位于项目目录test下
测试：
python runfinish.py <filename.md> <outputfilename>
<outputfilename>可以省略
```
